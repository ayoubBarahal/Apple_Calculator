# Apple Calculator

Welcome to the Apple Calculator project! This is a React-based calculator application inspired by the design and functionality of the Apple iOS calculator.

## Features

- Basic arithmetic operations: addition, subtraction, multiplication, and division
- Clear and backspace functions
- Responsive design, optimized for both desktop and mobile
